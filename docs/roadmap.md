# Roadmap

## v0.1.x hardening

- Add CI across macOS, Linux, and Windows.
- Add a first-class interactive permission confirmation channel.
- Add more provider-compatibility fixtures and session migration fixtures.
- Measure checkpoint write amplification and expose diagnostics without telemetry.
- Add optional OS sandbox launch profiles for plugins and shell/test processes.

## Deferred

Multi-agent orchestration, vector/embedding memory, browser automation, MCP, marketplace, web/TUI, accounts, collaboration, cloud sync, telemetry, analytics, schedulers, daemons, auto-push, auto-deploy, Kubernetes SDK integration, Go `.so` plugins, and a general workflow engine are intentionally not planned for v0.1.x. They require separate evidence and architecture decisions.
